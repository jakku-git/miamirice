# Menu Update Template for Miami Rice

Please fill out this template with your updated menu items and pricing. You can copy this format and fill it in, then return it to me.

## Instructions:
- Keep the same category structure (or let me know if you want to change categories)
- For each item, provide: Name, Description, Price
- Mark items as "Popular" by adding `popular: true` (or just mention it)
- For items with multiple price options, include all options in the price field
- If an item should be removed, just don't include it
- For drinks, use the subcategory structure shown below

---

## CATEGORY: Chef's Recommendations

```
- Name: [Item Name]
  Description: [Brief description of the dish]
  Price: $[XX.XX]
  Popular: [true/false - optional]
```

**Example:**
```
- Name: Rendang Beef
  Description: Slow-cooked beef in a rich, aromatic coconut sauce
  Price: $28.90
  Popular: true
```
